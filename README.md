# Credit_Default_Risk
Task:
Using the publicly available dataset Home Kaggle Credit (https://www.kaggle.com/c/home-credit-risk) start-up model of machine equipment servicing the backlog of costs. Justify the chosen methodology of building the model. Send us your presentation of results and justifications, and your R or Python code.

This project analyzes Home Credit data from Kaggle and builds Classification Algorithim using Random Forest Model to predict whether the creditor will default or not.
